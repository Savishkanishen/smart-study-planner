# Smart Study Planner

A JavaFX-based desktop application designed to help students track their syllabus, record performance scores, and automatically generate priority revision plans based on weak areas.

## Features
* **Custom Study Paths:** Add subjects and link prerequisites.
* **Syllabus Tracking:** Build an interactive tree structure for topics.
* **Performance Analytics:** Visual progress bars for topic and subject mastery.
* **Smart Revision Plan:** Algorithmically prioritizes weak topics (scores < 60%).

## Tech Stack
* **Frontend:** JavaFX (Clean Light Theme)
* **Backend:** Java 11+
* **Database:** MySQL
